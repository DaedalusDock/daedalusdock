Proc Details

New
(text, /atom/target, /mob/owner, /datum/language/language, /list/extra_classes, lifespan)

Constructs a chat message overlay

Arguments:

  • text - The text content of the overlay
  • target - The target atom to display the overlay at
  • owner - The mob that owns this overlay, only this mob will be able to view it
  • language - The language this message was spoken in
  • extra_classes - Extra classes to apply to the span that holds the text
  • lifespan - The lifespan of the message in deciseconds

proc
end_of_life
(fadetime)

Applies final animations to overlay CHAT_MESSAGE_EOL_FADE deciseconds prior to message deletion, sets timer for scheduling deletion

Arguments:

  • fadetime - The amount of time to animate the message's fadeout for
